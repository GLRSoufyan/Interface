Open de console en experimenteer met het volgende commando: typeof
Dit commando toont het datatype van de waarde die je invoert. Zie ook de voorbeelden hieronder. 

typeof "Hoi"
typeof 1
typeof []
typeof {a:[]}

String
  • 'Hoi'
  • ""
  • '1.2'
  • "5"
  • "{}"
  • '[]'
  • "function()"
  • Eigenlijk alles dat tussen " " of ' ' tekens staat
Number
  • 0
  • 1
  • 2.2
  • NaN
  • -10
  • 007
  • NaN staat voor Not A Number, maar het datatype van NaN is dus wel een number
Boolean
  • true
  • false
  • Boolean kan alleen maar true of false zijn
Null
  • null
  • Null vertegenwoordigt expliciet een lege of niet-bestaande waarde
Undefined
  • undefined
  • Undefined betekent dat een variabele wel is gedeclareerd, maar geen waarde heeft gekregen
Object
  • {}
  • { naam: 'Jan', leeftijd: 30 }
  • { href: '/index.html', target: '_blank' }
  • { naam: "Object", method: function() {} }
  • new Date()
  • Het JavaScript object datatype is het meest krachtige en daarmee complexe datatype, wat niet in een one-liner is toe te lichten. Check daarom deze link voor meer achtergrond informatie
Array
  • []
  • [1, 2, 3]
  • ['a', 'b', 'c']
  • [{ naam: 'Jan' }, { naam: 'Piet' }]
  • Een array is een lijst van waarden, het javascript datatype is een object.
Function
  • function() {}
  • function som(a, b) { return a + b; }
  • const functie = () => {}
  • Een functie is een blok code dat kan worden hergebruikt, het datatype is ook function
Symbol
  • Symbol()
  • const sym = Symbol('beschrijving')
  • Symbolen zijn unieke en onwijzigbare waarden, vaak gebruikt als unieke object-sleutels
BigInt
  • BigInt(1234567890123456789012345678901234567890)
  • 123n
  • BigInt is een datatype voor hele grote getallen die niet passen in het Number datatype
